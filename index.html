jq-strategy

聚宽（JoinQuant）量化策略代码生成器 · Claude Code Skill

用自然语言描述策略，Claude 自动生成可直接粘贴到 joinquant.com 回测的 Python 代码。

🚀 一键安装

curl -sL https://perfectuser21.github.io/jq-strategy-skill/install.sh | bash
装完重启 Claude Code，然后跟它说：
「写个策略，选市值最小的 3 只，每 5 天调仓」
Claude 会自动生成完整聚宽策略代码。

📦 手动下载

⬇ 下载 tar.gz ⬇ 下载 zip

解压后整个目录放到 ~/.claude/skills/jq-strategy/ 即可。

🎯 支持的策略类型

  • 选股轮动 —— 小市值、低估值、高 ROE、红利
  • 技术择时 —— 均线金叉、MACD、RSI
  • 多因子打分 —— 多指标综合排名
  • ETF 轮动 —— 动量切换、股债平衡

📋 触发方式（三种都行）

写个策略，选市值 20-30 亿，每 5 天换仓
/jq-strategy 破净股 + ROE>15，月度调仓持 10 只
帮我写个均线策略，510300 5 日线上穿 20 日线买入

🔧 国内加速（可选）

如果 GitHub 访问慢，可以用镜像：

git clone https://mirror.ghproxy.com/https://github.com/perfectuser21/jq-strategy-skill.git ~/.claude/skills/jq-strategy
